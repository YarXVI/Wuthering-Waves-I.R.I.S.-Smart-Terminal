# I.R.I.S. - Virtual Office - Build Instructions

## Quick Start (Recommended)
### Method 1: Direct Launcher (No Build Required)
Double-click to run:
```
启动.bat
```
Then select option 1 to start the development environment.
---

## Complete Build Instructions

### Prerequisites

- **Python 3.8+**
- **Node.js 18+**
- **npm**

### One-Click Full Build
Run directly:
```
构建.bat
```

This script will:
1. Install dependencies
2. Build frontend
3. Package backend
4. (Optional) Build Electron application

---

## Step-by-Step Build

### 1. Frontend Build

```bash
cd desktop
npm install
npm run build
```

### 2. Backend Package (Optional)

```bash
pip install pyinstaller
pyinstaller build_launcher.spec
```

Output: `dist/iris.exe`
